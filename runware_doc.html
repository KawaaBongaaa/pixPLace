üì§ Image Upload API

Request Format

POST /v1/ HTTP/1.1 Host: api.runware.ai Content-Type: application/json Authorization: Bearer YOUR_API_KEY { "taskType": "imageUpload", "taskUUID": "unique-task-identifier", "image": "base64-encoded-image-data" }

Parameters

taskType (string, required): Must be "imageUpload"
taskUUID (string, required): A unique identifier for this upload task
image (string, required): Base64 encoded image data (without data:image format prefix)

Authentication

Authorization: Bearer YOUR_API_KEY

Get your API key from the Runware dashboard at runware.ai/app/

Response Format

{ "data": { "taskType": "imageUpload", "imageUUID": "generated-uuid-here", "taskUUID": "your-task-uuid" } }

JavaScript Example

async function uploadToRunware(base64Image, apiKey) { const taskUUID = crypto.randomUUID(); const response = await fetch('https://api.runware.ai/v1/', { method: 'POST', headers: { 'Content-Type': 'application/json', 'Authorization': `Bearer ${apiKey}` }, body: JSON.stringify({ taskType: 'imageUpload', taskUUID: taskUUID, image: base64Image }) }); if (!response.ok) { throw new Error(`HTTP ${response.status}: ${response.statusText}`); } const result = await response.json(); return result.data.imageUUID; } // Usage example: const imageUUID = await uploadToRunware('iVBORw0KGgoAAAANS...', 'your-api-key');
