# pixPLace - AI Image Generator

Современное веб-приложение для генерации изображений с помощью ИИ, полностью переписанное на **Tailwind CSS**.

## 🎨 Дизайн-система

Приложение использует современную дизайн-систему на базе **Tailwind CSS 4.x** с следующими особенностями:

### ✨ Ключевые возможности
- **Glassmorphism эффекты** с backdrop-blur
- **Современные градиенты** и анимации
- **Адаптивный дизайн** (mobile-first)
- **Темная/светлая тема** с автоопределением
- **Консистентная цветовая палитра**
- **Микро-интеракции** и плавные переходы

### 🎯 Архитектура стилей

```
css/
├── main.css              # Основной файл с Tailwind директивами
├── components.css        # Переиспользуемые компоненты (формы, кнопки)
├── mode-cards-tailwind.css # Карточки режимов генерации
├── modals-tailwind.css   # Модальные окна и диалоги
└── shared/               # Специфические компоненты (legacy)
```

## 🛠️ Технический стек

### Frontend
- **Tailwind CSS 4.x** - утилитарный CSS фреймворк
- **PostCSS** - обработка CSS
- **Autoprefixer** - автоматические префиксы
- **Vanilla JavaScript (ES6+)** - модульная архитектура

### UI/UX компоненты
- Адаптивные карточки режимов
- Интерактивные модалы с анимациями
- Прогрессивные формы
- Glassmorphism эффекты
- Темная тема с системными предпочтениями

## 📱 Адаптивность

Приложение полностью адаптивно и оптимизировано для:

### 📱 Мобильные устройства (< 768px)
- Компактные карточки в 1 колонку
- Упрощенные контролы
- Touch-friendly интерфейс

### 📟 Планшеты (768px - 1024px)
- 2 колонки карточек
- Средние размеры элементов
- Оптимизированные отступы

### 💻 Десктопы (> 1024px)
- Полноценный интерфейс
- 3 колонки карточек
- Расширенные возможности
